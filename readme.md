# syncStreaming

mp3 및 mp4를 동시에 볼 수 있게 해주는 프로그램.

## 설치

- git clone https://github.com/namugach/syncStreaming.git
- npm init

## 실행

- shell: run.sh
- windows: run.bat

## 사용법
* 리스트 추가
  - src\public\content 안에 mp3, mp4 파일을 이동.
- 주의 사항
  - 리스트 추가 때 대소문자를 구분.
